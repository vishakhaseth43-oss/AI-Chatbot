#  Vishakha's AI Assistant

A simple web-based AI chatbot built using HTML, CSS, and JavaScript, powered by the OpenAI API. This project demonstrates how to create a conversational UI and connect it to an AI model.

---

##  Features

-  Interactive chat interface
-  Real-time AI responses
-  Clean and modern UI design
- Send messages using Enter key
- API integration with OpenAI
